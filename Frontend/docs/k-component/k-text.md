# [🔙](./index.md) k-text

k-text 用于展示文本

```html
<k-text size="large">This is text demo</k-text>
<k-text size="28" color="red">This is text demo</k-text>
```

## API

| 属性  | 类型                              | 可空 | 备注                |
| ----- | --------------------------------- | ---- | ------------------- |
| data  | string                            | 是   | 绑定的 data id      |
| color | string                            | 是   | 颜色                |
| size  | [Size](./types.md#size) \| number | 是   | 如果值为数字则为 px |
