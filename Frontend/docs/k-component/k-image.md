# [🔙](./index.md) k-image

k-text 用于展示图片

```html
<k-data>
  <k-data-item name="src">1.jpg</k-data-item>
</k-data>
<k-image data="src"></k-image>
<k-image data="1.jpg"></k-image>
```

## API

| 属性 | 类型   | 可空 | 备注                      |
| ---- | ------ | ---- | ------------------------- |
| data | string | 是   | 绑定的 data id 或图片路径 |
